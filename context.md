# Contexto do Projeto: Pineapple 🍍

## 🚀 Visão Geral
Este projeto é uma plataforma de venda de ingressos e gestão de eventos para a **Pineapple Party**, uma rave em Palotina. O foco é aplicar Engenharia de Software e IA para:
- Gestão de fluxo de entrada/saída com biometria facial (fotos).
- Análise de tendências de público (Gênero/Idade) para eventos futuros.
- Segurança de dados sensíveis (CPF, Tokens de Pagamento) usando a performance do Rust.

## 🛠️ Stack Tecnológica
- **Backend:** Rust (Rocket/Axum + Diesel/SQLx)
- **Frontend:** React + TypeScript (Foco em UX/UI e interatividades 3D)
- **Banco de Dados:** PostgreSQL
- **Segurança:** Gateway de Pagamento Externo (Tokenização)
